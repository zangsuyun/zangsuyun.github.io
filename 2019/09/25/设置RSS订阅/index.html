添加动态背景

Hexo Next 背景动画 Canvas_nest: true 设置无效的解决方案

解决方案

1.打开 Git 进入自己文件夹下/themes/hexo-theme-next 文件夹下

2.当前目录运行 git clone https://github.com/theme-next/theme-next-canvas-nest source/lib/canvas-nest （一定一定要当前目录下安装，自己安装不可以用的）

3.进入 thems/hexo-theme-next/_config.yml 文件下修改 canvas_nest: true (注意: 冒号后面要空格)

4.hexo s 本地查看效果
