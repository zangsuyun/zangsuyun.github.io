hexo上传git问题

发表于 2019-10-10 更新于 2019-10-13 阅读次数： Valine：

TypeError: Cannot set property ‘lastIndex’ of undefined

好久没更，本来好好的 hexo cl && hexo g && hexo d 上传博客，结果突然给我来这么一下，受不了受不了

解决方案一

修改配置文件_config.yml，注意不是主题里面的配置文件！
把 auto_detect 设置为 false，即可解决。

解决方案二

看到方案一的答案我有种莫名的亲切感，
auto_detect 自动判断代码语言
我想起在写上一篇关于 webpack 坑时候对代码块的引用语法上我故意少些了 bash 命令
加上bash果然上传成功

原因

auto_detect 设置为自动检测语言
github 源码地址使用说明
问题应该是由 highlight.js 尝试检测代码语言引起的。

主要就是 markdown 里对代码块的引用高亮显示问题,设置为 ture 却没有声明语言造成的
我在贴吧中看到说 所有在线文档都没有明确说明 markdown 代码块（```）到底支持哪些语法，
所以还是选择方案一，建议默认不要自动去识别语法，默认就全部不高亮，否则最后会出现莫名其妙的高亮。或者奇奇怪怪的问题
