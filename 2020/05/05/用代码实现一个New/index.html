用代码实现一个New

发表于 2020-05-05 更新于 2020-12-16 阅读次数： Valine：

如何理解 [].shift.call(arguments)
因为 shift 内部实现是使用的this代表调用对象。那么当[].shift.call() 传入 arguments对象的时候，
通过 call函数改变原来 shift 方法的 this 指向, 使其指向 arguments，并对 arguments 进行复制操作，
而后返回一个新数组。至此便是完成了 arguments 类数组转为数组的目的！其实这可以理解为，让类数组调用数组的方法！
